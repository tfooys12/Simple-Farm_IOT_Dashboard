# 🌱 Farm IoT Dashboard

## 📌 Project Description
This project is a simple Farm IoT Dashboard that simulates real-time monitoring of farm conditions such as temperature and soil moisture. It uses JavaScript to generate live data and visualize it on a web dashboard.
